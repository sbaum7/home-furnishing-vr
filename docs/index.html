Project Overview

This VR game allows players to arrange furniture in virtual rooms (Living Room, Bedroom, Kitchen) using XR Interaction Toolkit. Players select difficulty and hint options from a start menu, spawn furniture via radial menus or tabs, and place items into designated sockets. Levels are completed when all items are correctly placed, triggering confetti and progression to the next scene.

Key features:

  • Difficulty selection with/without hints
  • Radial menu spawning with floor-clipping prevention
  • Socket-based placement validation
  • Confetti celebration on level completion
  • Pause menu and physics restoration on release

Detailed Class Documentation

PauseMenu

Handles the in-game pause menu functionality, including resuming the game and quitting to a specified scene.

LivingRoomLevelManager

Manages level completion logic for the Living Room scene by checking correct placement of specific furniture items.

PlacementChecker

Monitors a single XR Socket for correct item placement and notifies the PlacementManager.

ItemId

Simple component that assigns a unique string identifier to furniture items for validation.

RestorePhysics

Restores original physics settings (gravity/kinematic) after an object is released from grabbing.

PlacementManager

Central manager that tracks all socket placements, triggers confetti, and loads the next scene on completion.

FurnitureSpawner

Utility for spawning and replacing a single furniture item at a fixed location.

StartMenuManager

Manages the main menu UI: difficulty selection, hints toggle, and scene loading.

RoomTabsMenu

VR radial/tab menu toggled by controller input; spawns furniture with offset to avoid floor clipping.

FurniturePager

Spawns an entire category of furniture items at predefined points when a button is pressed.

StudyStartButton

Simple button handler that loads a configured scene.

SimpleSpawn

Quick testing utility for spawning furniture by index (originally in TabsControl.cs).
